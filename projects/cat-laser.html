Cat Laser Tower

Often, my cat's enthusiasm for 'chase the laser' continues long after my own enjoyment of 'look how silly the cat is' has ended, so I turned to technology to find an answer.

The Arduino powered Cat Laser Tower (which can double as a scratching post) zips a red dot randomly around the floor, allowing you to sit back and enjoy the ridiculous antics of your feline friend.

I don't know who had more fun, the cat, or everyone watching the cat.

This project uses a laser and lasers can be dangerous. As we're using one which is no more powerful than a standard laser pointer, the blink reflex for both cats and humans should prevent any serious damage to the eyes.

This is a super simple project, using only an Arduino Nano, a laser module and a servo for the electronics. It was one of the first Arduino projects I built and was a great introduction to hobbyist electronics.
